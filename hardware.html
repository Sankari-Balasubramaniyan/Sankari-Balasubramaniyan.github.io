LIST OF HARDWARE PROJECTS

ROS Robot movement

Using the meta operating system ROS, implemented an algorithm to establish TURTLEBOT3 movement. The robot was programmed using python to choose a straight line trajectory and avoid obstacle to reach a goal in the given simulated world.

Contactless Digital Tachometer

A three digit contact less digital tachometer using 8051 microcontroller which can be used for measuring the revolutions per second of a rotating wheel, disc, shaft, or anything like that.

Ardino Security and Alarm system

An alarm system designed which detects objects using ultrasonic sensor and can be put to off using a 4 digitpassword.

Garbage Monitoring system

Implemented a smart bin built on a microcontroller-based platform Arduino Uno board which is interfaced with Wifi module and Ultrasonic sensor which can gives the status of the waste present in the dustbin to the municipal authority

Urban road transport system

This project presents Road transport system which consist of two different implementations and presents the idea of automated streetlight system and traffic light control based on density and thereby enables efficient usage of our time and energy.
